# Weather App

A simple weather application built with HTML, CSS, and JavaScript.

Users can search for any city and view live weather information using the Visual Crossing Weather API.

---

## Features

- Search weather by city
- Fetch live weather data from an API
- Display:
  - Temperature
  - Weather conditions
  - Humidity
  - Wind speed
- Loading state while fetching data
- Error handling
- Responsive and styled UI

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- Visual Crossing Weather API
